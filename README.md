
# Restaurant_Website  

## Description  
**Restaurant_Website** is a responsive landing page designed for a restaurant. This project highlights my skills in working with React, JavaScript, HTML, and CSS. It showcases a clean and optimized design while maintaining a logical and organized structure using the **BEM (Block Element Modifier)** methodology for CSS.  

The primary focus of this project was to ensure:  
- Responsiveness across all devices.  
- Code readability and maintainability for other developers.  
- Optimized performance for a smooth user experience.  

---

## Features  
- **Responsive Design**: Seamlessly adapts to desktop, tablet, and mobile devices.  
- **Organized CSS with BEM**: Ensures logical and modular styling for easy maintenance.  
- **Optimized Performance**: Designed with best practices to ensure quick loading and smooth interaction.  

---

## How to Use  

### Installation  
1. Clone this repository:  
   ```bash  
   git clone https://github.com/behnamac/project_restaurant_website.git  
   ```  

2. Navigate to the project directory:  
   ```bash  
   cd project_restaurant_website  
   ```  

3. Install dependencies:  
   ```bash  
   npm install  
   ```  

### Development  
Start the development server:  
   ```bash  
   npm run dev  
   ```  

### Build  
Build the project for production:  
   ```bash  
   npm run build  
   ```
